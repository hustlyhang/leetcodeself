- 5849. 好子集的数目
    - 求最大公约数
        ```C++
        int gcd(int a, int b) {
            return b ? gcd(b, a % b) : a;
        }
        ```
    - 去除不能选的点
        ```C++
        // 去除不能选的点
        for (int i = 2; i * i <= 30; ++i) {
            for (int j = 1; j * i * i <= 30; ++j) {
                st[j * i * i] = 1;
            }
        }
        // 平方和平方的倍数去除掉
        ```
    - 使用图来保存两个数之间是否互质，便于后面判断，互质是根据最大公约数来进行判断的
